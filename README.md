
# Зарплатный календарь — модульная сборка

## Как запустить
```bash
python3 run.py
```
(требуется Python 3.9+; внешних зависимостей нет — используется стандартная библиотека и tkinter)

## Структура
- `run.py` — точка входа, запускает GUI.
- `salary_calendar/__init__.py` — файл пакета.
- `salary_calendar/constants.py` — константы (цвета, ставки), денежные/временные утилиты.
- `salary_calendar/database.py` — работа с SQLite: создание БД, CRUD для `shifts`, поисковые запросы.
- `salary_calendar/calculations.py` — все расчёты (рабочие дни, ставка за час, переработка, выходные).
- `salary_calendar/events.py` — сервисные операции: добавление доплаты к дню, распределение переработки по недоработкам в половине месяца.
- `salary_calendar/widgets.py` — виджеты Tk: тултип и диалог редактирования смены.
- `salary_calendar/export.py` — экспорт CSV (UTF-8 с BOM, разделитель `;` по умолчанию).
- `salary_calendar/interface.py` — основной GUI: календарь, раскраска, недельные итоги, кнопки управления, показ ожидаемого конца смены, расчёт зарплат 14/29, обработка переработок, экспорт.

## Совместимость
- База лежит в файле `salary_calendar.db` (в рабочем каталоге).
- Экспорт создаёт CSV вида `salary_export_YYYY-MM-DD.csv`. Excel корректно распознаёт UTF-8 благодаря BOM.
- Функциональность идентична монолитной версии, различия только в разнесении кода по модулям и мелкой чистке.
